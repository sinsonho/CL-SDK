# 🧠 살아있는 컴퓨터를 코딩하다

### 하드웨어 없이 CL-SDK로 배우는 파이썬 코딩

*실리콘 너머, 살아있는 뉴런에게 코드를 보내다*

## 📖 책 소개

값비싼 하드웨어나 웻랩(wet lab) 없이, **파이썬과 CL-SDK 시뮬레이터만으로** '살아있는 컴퓨터'를 직접 다뤄보는 실전 입문서입니다.

실제로 살아있는 뉴런에게 연산을 맡기는 **바이오컴퓨팅**의 세계를, 이제는 실험실 장비 없이 코드만으로 발 들일 수 있습니다. 이 책은 바로 그 출발선입니다.

---

## 🤔 왜 이 책인가

현장에서 매일 마주하는 벽이 있습니다.

지금의 컴퓨터는 메모리와 연산이 분리된 구조(**폰노이만 병목**)와 **발열·전력**이라는 물리적 한계에 부딪혀 있습니다. 더 빠르게, 더 똑똑하게 가려는 모든 시도가 결국 이 두 벽 앞에서 비용을 치릅니다.

그 너머를 탐구하는 가장 급진적인 답 중 하나가 바이오컴퓨팅 — 실제로 살아있는 뉴런에게 연산을 맡기는 것입니다.

---

## 📚 이 책에서 만나는 것

| 구분 | 내용 |
|------|------|
| 🚀 **시작** | 하드웨어 없이, 파이썬 환경만 있으면 됩니다 |
| 🧠 **규모** | 80만 개의 살아있는 뉴런 |
| 🔄 **핵심 루프** | 신호 전송(`stim`) → 반응 측정(`read`) → 학습(`loop`) |
| ⚡ **작동 원리** | 실리콘과 시냅스를 잇는(silicon ↔ synapse) 코드 |
| 🌐 **패러다임** | 폰노이만 병목과 발열 너머의 차세대 컴퓨팅 |

---

## 🎯 이런 분께 권합니다

- 다음 세대의 컴퓨팅이 어디로 향하는지 궁금한 **개발자**

- 코드로 직접 '살아있는 시스템'을 만져보고 싶은 **모든 분**
